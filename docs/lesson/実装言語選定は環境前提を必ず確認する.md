# 実装言語選定は環境前提を必ず確認する

**カテゴリ**: lesson  
**タグ**: implementation,python,sqlite,2026-04-10,preflight  
**登録日**: 2026-04-10 14:59:10  

---

Phase 5 で sqlite3 CLI を Bash から呼ぶ計画だったが、Windows Git Bash 環境に sqlite3 が存在せず実行不能だった。Python 3.13 (sqlite3 + FTS5 標準同梱) に切り替えて対応。

学び:
1. CLI 言語選定時、環境にツールが入っているかを必ず事前確認する（preflight check）
2. Python は標準ライブラリで SQLite + FTS5 が完全に使える。Windows でも追加インストール不要
3. 副次効果: Phase 4 Round 2 で懸念していた『Bash CLI の引用符地獄・特殊文字の脆弱性』が構造的に解決された
4. ユーザーが指定した方針（『Bash でいい』）は前提条件付き。前提が崩れたら再確認する権利がある
